# Reglas ADAMS para importación a Odoo

Columnas de `product_product.csv`:
- External ID
- Name
- Product Type
- Internal Reference
- Barcode
- Sales Price
- Cost
- Weight
- Sales Description
- Product Values

Columnas de `initial_stock.csv`:
- sku
- ubicacion_odoo
- cantidad
- costo_unitario

Ubicaciones:
- BPU/Existencias
- TUR/Existencias
- BAT/Existencias
